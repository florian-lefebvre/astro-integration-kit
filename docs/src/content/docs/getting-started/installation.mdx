import { Tabs, TabItem } from '@astrojs/starlight/components';

Add `astro-integration-kit` using your favorite package manager:

<Tabs>
    <TabItem label="npm">
    ```bash
    npm install astro-integration-kit
    ```
    </TabItem>
    <TabItem label="yarn">
    ```bash
    yarn add astro-integration-kit
    ```
    </TabItem>
    <TabItem label="pnpm">
    ```bash
    pnpm add astro-integration-kit
    ```
    </TabItem>
</Tabs>

You can now start using our powerful utilities:

```ts
import { defineIntegration } from "astro-integration-kit"
```

Head to the next section to learn how!
